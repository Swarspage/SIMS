As there is protected routing follow this -


For access of admin pages type this in console - 

localStorage.setItem("token", "fake-admin-token");
localStorage.setItem("role", "admin");
goto ----> http://localhost:5173/admin/dashboard


then to access student dashboard 
type this in console -

localStorage.clear();


then

localStorage.setItem("token", "fake-student-token");
localStorage.setItem("role", "student");

goto ---------> http://localhost:5173/student/dashboard
